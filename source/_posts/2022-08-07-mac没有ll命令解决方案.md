# mac没有ll命令解决方案

### 1. 编辑.bash_profile文件


```
vim ~/.bash_profile
```

### 2. 在bash_profile中添加：

```
alias ll='ls -alF'
```

### 3. 最后让修改立即生效

```
source ~/.bash_profile
```

### 4. 根目录下创建.zshrc 文件

~~~
touch .zshrc
~~~

### 5. .zshrc文件 新增内容：

~~~
source ~/.bash_profile
~~~

### 6. 退出文件，同级目录下重新执行：

~~~
source .zshrc
~~~
